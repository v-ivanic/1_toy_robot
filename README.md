# Toy robot game

## Running the game
Start the game with

terminal> python main.py

Arrange sequence of commands (separated by space) in the following form:
* PLACE X,Y,F
* MOVE
* LEFT
* RIGHT
* REPORT

X and Y should be between 0 and 4. F must be NORTH, SOUTH, EAST or WEST.
